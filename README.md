# 계산기 프로그램
개발완료 : 2026년 1월 13일<br>
사용언어 : Python<br>
개발목적 : 사직연산을 한번에 해주는 프로그램이 있으면 좋겠다는 생각이 들어서 개발을 시작했습니다.<br>
사용방법 : 계산하고 싶은 두 수를 입력하면 사칙연산(+-*/) 결과가 나옵니다.<br>

# 나이 계산 프로그램
개발완료 : 2026년 1월 13일<br>
사용언어 : Python<br>
개발목적 : 나이 계산이 힘든 분들을 위해서 현재년도와 태어난년도를 입력하면 현재 나이를 알수 있도록 개발했습니다..<br>
사용방법 : 먼저 현재년도를 입력하고 태어난년도를 입력하면 현재 나이가 나옵니다.<br>
